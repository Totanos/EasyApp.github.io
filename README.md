# EasyApp GitHub Pages

Statische Website fuer `Totanos/EasyApp.github.io`.

## Dateien

- `index.html` ist die oeffentliche Startseite.
- `styles.css` enthaelt das responsive Layout.
- `script.js` setzt nur das Jahr im Footer.
- `.nojekyll` sorgt dafuer, dass GitHub Pages die Dateien unveraendert ausliefert.

## GitHub Pages

Wenn Pages noch nicht aktiv ist:

1. Repository `Settings` oeffnen.
2. `Pages` auswaehlen.
3. Unter `Build and deployment` die Quelle `Deploy from a branch` waehlen.
4. Branch `main` und Ordner `/ (root)` auswaehlen.

Danach ist die Projektseite normalerweise unter `https://totanos.github.io/EasyApp.github.io/` erreichbar.
